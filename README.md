# sb-order-service

`sb-order-service` is a springboot based microservice, designed to manage customer orders at scale.
It follows Hexagonal Architecture (a.k.a. Usecases, Ports & Adapters) to ensure clear separation of concerns, testability and flexibility in integrating with external systems.

### build and run...

**mvn clean install**

Refer **OrderServiceApplication**.java to run the application.
